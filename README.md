本目录包含10个专业的Python爬虫程序，覆盖不同的网站和数据类型，每个爬虫都有完整的功能、错误处理和详细注释。

## 功能特点

### 1. 完整的功能实现
- 数据爬取：支持多种数据类型的爬取
- 数据解析：使用BeautifulSoup、正则表达式等解析HTML/JSON
- 数据存储：支持JSON、CSV格式保存
- 分页处理：支持自动翻页和批量获取

### 2. 完善的错误处理
- 网络异常处理：连接超时、请求失败等
- 数据解析异常：HTML结构变化、JSON解析失败等
- 重试机制：指数退避重试策略
- 缓存机制：避免重复请求

### 3. 数据分析功能
- 数据清洗：去除HTML标签、清理无效数据
- 指标计算：热度评分、质量评分、互动率等
- 统计分析：数据分布、趋势分析等

### 4. 代码质量
- 详细注释：每个函数都有完整的文档字符串
- 类型提示：使用Python类型提示
- 模块化设计：类和方法设计合理
- 配置灵活：支持自定义请求头、代理等

## 使用说明

### 环境要求
- Python 3.7+
- 依赖包：
  ```
  requests
  beautifulsoup4
  lxml
  ```

### 安装依赖
```bash
pip install requests beautifulsoup4 lxml
```

### 运行单个爬虫
```bash
# 直接运行
python crawler_41.py

# 或导入使用
from crawler_41 import WeiboTrendingCrawler
crawler = WeiboTrendingCrawler()
data = crawler.run(save_to_file=True)
```

### 运行所有爬虫测试
```bash
python test_crawlers.py
```

## 爬虫详细说明

### 1. 微博热搜爬虫 (crawler_41.py)
- 功能：获取微博实时热搜榜
- 数据：热搜关键词、热度值、分类标签
- 扩展：可获取热搜详细微博内容
- 保存：JSON格式

### 2. 抖音网红数据爬虫 (crawler_42.py)
- 功能：搜索抖音网红账号
- 数据：网红基本信息、粉丝数、作品数
- 扩展：可获取网红详细资料和视频数据
- 分析：互动率、内容质量评分
- 保存：CSV格式

### 3. B站视频爬虫 (crawler_43.py)
- 功能：获取B站热门视频
- 数据：视频信息、UP主信息、统计数据
- 扩展：支持分区视频获取
- 分析：互动率、质量评分、热度评分
- 保存：JSON格式

### 4. 斗鱼直播爬虫 (crawler_44.py)
- 功能：获取斗鱼热门直播间
- 数据：直播间信息、主播信息、在线人数
- 扩展：可收集弹幕数据（简化版）
- 分析：热度评分、互动率
- 保存：CSV格式

### 5. 京东评论爬虫 (crawler_45.py)
- 功能：搜索京东商品并获取评论
- 数据：商品信息、评论内容、评分
- 分析：评分分布、关键词分析
- 保存：JSON格式

### 6. 豆瓣电影爬虫 (crawler_46.py)
- 功能：获取豆瓣电影信息和评论
- 数据：电影详情、评分、评论
- 扩展：支持Top250电影获取
- 分析：评分分布、情感分析
- 保存：CSV格式

### 7. 知乎热榜爬虫 (crawler_47.py)
- 功能：获取知乎热榜问题
- 数据：问题信息、回答内容、用户信息
- 分析：回答质量、专业度分析
- 保存：JSON格式

### 8. Stack Overflow爬虫 (crawler_48.py)
- 功能：获取Stack Overflow热门问题
- 数据：问题信息、回答内容、用户信息
- 扩展：支持API调用
- 分析：回答质量、采纳率
- 保存：CSV格式

### 9. GitHub Trending爬虫 (crawler_49.py)
- 功能：获取GitHub Trending仓库
- 数据：仓库信息、语言统计、用户信息
- 分析：活跃度、流行度、健康度评分
- 保存：JSON格式

### 10. V2EX社区爬虫 (crawler_50.py)
- 功能：获取V2EX热门话题
- 数据：话题信息、回复内容、用户信息
- 扩展：支持节点话题获取
- 分析：热度评分、质量评分
- 保存：JSON格式

## 配置说明

### 请求头配置
每个爬虫都支持自定义请求头，可修改User-Agent等参数避免被屏蔽。

### 代理配置
支持HTTP/HTTPS代理，可在初始化时传入：
```python
crawler = WeiboTrendingCrawler(proxy={'http': 'http://proxy:port', 'https': 'https://proxy:port'})
```

### 数据保存
数据默认保存到 `data/` 目录，文件名包含时间戳。

## 注意事项

### 1. 遵守robots.txt
- 请遵守目标网站的robots.txt规则
- 合理设置请求间隔，避免对服务器造成压力

### 2. 频率限制
- 每个爬虫都有随机延迟，避免请求过快
- 建议在实际使用时根据需要调整延迟时间

### 3. 数据使用
- 爬取的数据仅用于学习和研究
- 请勿用于商业用途或侵犯他人权益

### 4. 网站更新
- 网站结构可能随时变化，需要定期更新解析逻辑
- 每个爬虫都有错误处理机制，便于调试
