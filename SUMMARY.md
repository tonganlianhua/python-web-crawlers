## 使用说明

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行单个爬虫
```bash
python crawler_41.py
```

### 测试所有爬虫
```bash
python test_crawlers.py
```

### 导入使用
```python
from crawler_41 import WeiboTrendingCrawler
crawler = WeiboTrendingCrawler()
data = crawler.run(save_to_file=True)
```

## 技术栈
- **爬虫框架**: 原生requests + BeautifulSoup
- **数据解析**: BeautifulSoup, re (正则表达式), json
- **数据存储**: JSON, CSV
- **错误处理**: try-except, 重试机制
- **性能优化**: 请求缓存, 延迟控制
