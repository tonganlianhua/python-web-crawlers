# Python爬虫程序集合 (crawler_21.py - crawler_30.py)

## 概述

这10个Python爬虫程序覆盖了不同的网站和数据类型，每个爬虫都有完整的功能、错误处理和详细注释。所有程序都可以实际运行，并具有良好的代码质量。

## 爬虫列表

### 1. crawler_21.py - B站视频数据爬虫
**网站**: 哔哩哔哩 (Bilibili)
**功能**:
- 爬取B站热门视频信息
- 获取视频标题、播放量、弹幕数、UP主等数据
- 支持获取视频详细信息
- 数据保存为CSV和JSON格式
**特点**: 完整的API调用，错误处理，数据格式化

### 2. crawler_22.py - 网易云音乐爬虫
**网站**: 网易云音乐 (Netease Cloud Music)
**功能**:
- 获取热歌榜歌曲列表
- 爬取歌曲名、歌手、专辑、时长、播放量等
- 获取歌曲评论
- 数据分析功能
**特点**: 模拟API加密，复杂的数据解析，评论功能

### 3. crawler_23.py - Steam游戏数据爬虫
**网站**: Steam游戏平台
**功能**:
- 获取热门游戏信息
- 爬取游戏名、价格、评分、发行日期、标签等
- 搜索游戏功能
- 游戏详情获取
**特点**: 多语言支持，折扣信息，游戏标签分析

### 4. crawler_24.py - NBA数据爬虫
**网站**: ESPN NBA数据
**功能**:
- 获取NBA比赛赛程和比分
- 爬取球队排名和战绩
- 获取球员统计数据
- 支持赛程分析和统计
**特点**: 实时比赛状态，多类型数据，球队映射

### 5. crawler_25.py - 丁香医生健康资讯爬虫
**网站**: 丁香医生健康平台
**功能**:
- 爬取健康科普文章
- 获取文章标题、作者、分类、发布时间等
- 文章内容提取
- 搜索功能
**特点**: HTML解析，内容清理，健康分类系统

### 6. crawler_26.py - 下厨房菜谱爬虫
**网站**: 下厨房美食平台
**功能**:
- 获取热门菜谱
- 爬取菜谱名、食材、做法、评分等
- 根据食材搜索菜谱
- 菜谱详情获取
**特点**: 食材解析，步骤提取，烹饪时间分析

### 7. crawler_27.py - 携程旅游景点爬虫
**网站**: 携程旅游 (Ctrip)
**功能**:
- 获取城市旅游景点信息
- 爬取景点名称、评分、门票价格、开放时间等
- 推荐旅游路线
- 景点搜索功能
**特点**: 位置信息，最佳季节，旅游路线规划

### 8. crawler_28.py - 链家二手房数据爬虫
**网站**: 链家房地产
**功能**:
- 获取二手房信息
- 爬取房价、面积、户型、楼层、朝向等
- 支持多城市搜索
- 房屋筛选功能
**特点**: 价格分析，位置解析，房屋详情获取

### 9. crawler_29.py - 汽车之家车型数据爬虫
**网站**: 汽车之家 (Autohome)
**功能**:
- 获取汽车车型信息
- 爬取车型参数、价格、配置、评分等
- 车型对比功能
- 筛选搜索功能
**特点**: 参数解析，品牌识别，车型对比

### 10. crawler_30.py - 京东电子产品价格爬虫
**网站**: 京东商城 (JD.com)
**功能**:
- 获取电子产品价格信息
- 爬取商品价格、品牌、评分、销量等
- 价格历史模拟
- 商品筛选功能
**特点**: 价格监控，规格解析，促销信息

## 运行环境要求

### 基本要求
- Python 3.7+
- requests库
- BeautifulSoup4 (用于HTML解析)
- 可选: pandas (用于数据分析)

### 安装依赖
```bash
pip install requests beautifulsoup4
```

## 使用方法

### 直接运行
```bash
python crawler_21.py
python crawler_22.py
...
```

### 配置选项
每个爬虫都支持命令行交互，可以选择：
- 分类/类型
- 城市/地区
- 关键词搜索
- 筛选条件

### 数据输出
所有爬虫都会将数据保存为：
- CSV文件 (UTF-8编码，支持中文)
- JSON文件 (格式化输出)
- 控制台统计信息
